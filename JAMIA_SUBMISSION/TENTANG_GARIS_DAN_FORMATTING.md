# Tentang Garis Horizontal dan Formatting untuk JAMIA

## ‚ùì PERTANYAAN: Apakah Garis Diperbolehkan?

### Garis Horizontal (Horizontal Rules/Lines)

#### ‚ùå TIDAK DIREKOMENDASIKAN untuk Jurnal Ilmiah:

**JAMIA dan kebanyakan jurnal medis TIDAK mengizinkan:**
- Garis dekoratif horizontal
- Border atau frames di sekitar teks
- Fancy separators
- Ornamental lines

**ALASAN:**
1. Jurnal memiliki style guide sendiri untuk publikasi
2. Garis akan dihapus saat typesetting
3. Bisa mengganggu proses editorial
4. Tidak sesuai format akademik

## üîß CARA MENGHAPUS GARIS (Jika Ada)

### Jika ada garis horizontal di dokumen Word:

#### Method 1: Delete Border
1. Klik line/garis yang mau dihapus
2. Tab **Home** > **Borders** dropdown
3. Pilih **No Border**

#### Method 2: Delete Horizontal Line
1. Letakkan cursor di atas atau di bawah garis
2. Tab **Home** > **Borders** dropdown > **Borders and Shading**
3. Pilih **None**
4. Klik **OK**

#### Method 3: Delete All Borders
1. Select all (Ctrl+A)
2. Tab **Design** (atau **Page Layout**)
3. **Page Borders** > pilih **None**
4. Apply to **Whole document**

## üìù FORMATTING YANG BENAR UNTUK JAMIA

### Manuscript Structure (TANPA Garis):

```
TITLE (Heading 1, Bold, Centered)
[blank line]
Authors (Regular text, Centered)
[blank line]
Affiliations (Regular text, Centered)
[blank line]
[blank line]
ABSTRACT (Heading 1)
[blank line]
Abstract text...
[blank line]
Keywords: keyword1, keyword2, keyword3
[blank line]
[blank line]
INTRODUCTION (Heading 1)
[blank line]
Introduction text...
[blank line]
[blank line]
MATERIALS AND METHODS (Heading 1)
[blank line]
Methods text...
```

### Gunakan Heading Styles:
- **Heading 1**: ABSTRACT, INTRODUCTION, METHODS, RESULTS, DISCUSSION, CONCLUSION
- **Heading 2**: Sub-sections (Dataset, Models, Evaluation)
- **Heading 3**: Sub-sub-sections (jika perlu)

### JANGAN Gunakan:
- ‚ùå Horizontal lines/rules
- ‚ùå Text boxes
- ‚ùå Shapes/drawing objects
- ‚ùå Background colors
- ‚ùå Fancy fonts
- ‚ùå Multiple columns (kecuali references jika diminta)

## üéØ CHECKLIST FORMATTING BERSIH

Sebelum submit, pastikan dokumen BERSIH:

- [ ] Tidak ada garis horizontal dekoratif
- [ ] Tidak ada text boxes
- [ ] Tidak ada background colors
- [ ] Hanya menggunakan heading styles standar
- [ ] Font konsisten (Times New Roman 12pt atau Arial 11pt)
- [ ] Spacing konsisten (double-spaced atau 1.5)
- [ ] Margins 1 inch semua sisi
- [ ] Tabel menggunakan format sederhana
- [ ] Tidak ada track changes atau comments

## üí° TIPS

### Untuk membuat dokumen bersih:

1. **Gunakan Styles** (Heading 1, 2, 3) bukan format manual
2. **Konsisten**: Gunakan spacing yang sama di seluruh dokumen
3. **Sederhana**: Keep it simple - fokus pada konten bukan dekorasi
4. **Cek guidelines**: Setiap jurnal punya requirements sendiri

## ‚ùì KESIMPULAN

**Q: Apakah garis horizontal diperbolehkan?**
**A: TIDAK untuk garis dekoratif. YA untuk border tabel standar.**

**Q: Bagaimana memisahkan sections?**
**A: Gunakan heading styles dan blank lines, BUKAN garis.**

**Q: Apa yang harus saya lakukan dengan garis yang sudah ada?**
**A: Hapus semua garis dekoratif. Gunakan formatting bersih.**
