### Method 1: Automatic Line Numbers (RECOMMENDED)

#### Step 1: Buka manuscript.docx
- Klik kanan **Converted/manuscript.docx**
- Pilih **Open with Microsoft Word**

#### Step 2: Tambahkan Line Numbers
1. Klik tab **Layout** (atau **Page Layout** di Word versi lama)
2. Di grup **Page Setup**, klik **Line Numbers**
3. Pilih **Line Numbering Options...**
4. Akan muncul dialog **Page Setup**
5. Klik tombol **Line Numbers...** (di bagian bawah)
6. Centang âœ… **Add line numbering**
7. Pilih pengaturan:
   - **Start at:** 1
   - **From text:** Auto
   - **Count by:** 1 (setiap baris diberi nomor)
   - **Numbering:** âœ… **Continuous** (pilih ini!)
     - Atau bisa pilih **Restart each page** jika jurnal minta
8. Klik **OK**
9. Klik **OK** lagi

#### Step 3: Verifikasi
- Scroll dokumen dan pastikan nomor baris muncul di margin kiri
- Nomor akan muncul di setiap baris teks
- Nomor TIDAK akan muncul di baris kosong atau heading (ini normal)

#### Step 4: Simpan
- **File > Save As**
- Atau **Ctrl+S**

## âš ï¸ TROUBLESHOOTING

### Line numbers tidak muncul?
- **Cek View**: Pastikan dalam **Print Layout** view (bukan Web Layout)
- **Cek Section**: Line numbers apply per section. Jika ada section breaks, perlu set di tiap section

### Line numbers muncul di header/footer?
- Ini tidak seharusnya terjadi. Cek Page Setup > Line Numbers > uncheck jika ada

### Line numbers restart di tiap halaman?
- Layout > Line Numbers > Line Numbering Options
- Pastikan pilih **Continuous** bukan "Restart each page"

## ðŸ“„ PAGE NUMBERS (Juga Wajib)

### Cara Menambahkan Page Numbers:

1. Klik tab **Insert**
2. Klik **Page Number**
3. Pilih **Bottom of Page** > **Plain Number 2** (centered) atau **Plain Number 3** (right)
4. Atau sesuai preferensi jurnal
5. Klik di luar header/footer area untuk keluar dari editing mode
